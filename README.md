# 🎬 React Movie App

A responsive movie browsing application built with **React.js** and **Tailwind CSS** that allows users to discover, search, and explore movies.

---

## 🚀 Live Demo

[View Live App](https://react-movie-app-two-lac.vercel.app)

---

## ✨ Features

- 🔍 Search movies by title
- 🎥 Browse popular and trending movies
- 📋 View detailed movie information and ratings
- 📱 Fully responsive design — works on mobile and desktop
- ⚡ Fast and smooth UI with React

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js | Frontend framework |
| Tailwind CSS | Styling and responsive design |
| JavaScript (ES6+) | Application logic |
| Movie Database API (TMDB) | Movie data source |
| Vite | Build tool |
| Vercel | Deployment |

---

## ⚙️ How to Run Locally

**Prerequisites:** Node.js installed on your machine

```bash
# Clone the repository
git clone https://github.com/tech-deb/React-Movie-App.git

# Navigate into the project directory
cd React-Movie-App

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 🔧 Environment Variables

Create a `.env` file in the root directory and add:

```
VITE_API_KEY=your_tmdb_api_key_here
```

Get your free API key from [The Movie Database (TMDB)](https://www.themoviedb.org/settings/api)

## 🚀 Deployment

This app is deployed on **Vercel**. To deploy your own version:

1. Fork this repository
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Add your environment variables
5. Click Deploy
